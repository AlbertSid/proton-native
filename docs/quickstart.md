# Quickstart

## Prerequisites

### Linux

`qtbase5-dev`

## Install

### Automatic

```bash
# install the cli app
npm install -g create-proton-app
# create your project
create-proton-app my-app
# move to your project directory
cd my-app
# run your app
npm run start
```

You can alternatively use `npx` if you prefer.

The old manual instructions are still available [here](manual_install.md)
