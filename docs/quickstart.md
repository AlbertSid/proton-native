# Quickstart

## Prerequisites

### Linux

`qtbase5-dev`

## Install

### Automatic

```bash
# install the cli app
npm install -g proton-native-cli
# create your project
proton-native init my-app
# move to your project directory
cd my-app

# run your app
npm run start
# OR to run with hot reloading
npm run dev
```

You can alternatively use `npx` if you prefer.
